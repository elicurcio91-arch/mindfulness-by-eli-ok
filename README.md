# MindfulnessbyEli

Sitio web oficial de MindfulnessbyEli. Una plataforma dedicada a encontrar más alegría en la vida a través del mindfulness.

## Desarrollo Local

**Prerrequisitos:** Node.js

1. Instalar dependencias:
   `npm install`
2. Iniciar el servidor de desarrollo:
   `npm run dev`

## Tecnologías

- React 19
- Vite
- Tailwind CSS
- TypeScript
